Simple Sync Test

This page tests the new server-first approach for data synchronization.

Test Overview

The new approach is much simpler:

  • Server is the single source of truth
  • All changes go through the server immediately
  • Client always reflects server state
  • No complex merging or conflict resolution

API Endpoints

The new API endpoints are:

  • GET /api/user/data - Get current server state
  • POST /api/user/children - Create/update/delete children
  • POST /api/user/goals - Create/update/delete goals
